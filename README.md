# vue-ssr-starter

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Start server
```
npm run start
```

See [Vue SSR Guide](https://ssr.vuejs.org/)
